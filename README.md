# 🏡 Aplikasi Manajemen Desa

Sistem informasi desa berbasis web untuk memudahkan pengelolaan data desa secara digital.

## ✨ Fitur
- Dashboard admin dengan statistik real-time
- Manajemen data warga
- Data kegiatan desa
- Data keuangan
- Partisipasi warga
- Data perangkat desa
- Laporan kegiatan

## 🛠️ Teknologi
- PHP
- MySQL
- HTML & CSS
